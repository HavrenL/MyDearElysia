# 参考音频命名示例

将你的参考音频放在此目录下，按以下格式命名：

```
【情绪】音频文本内容.wav
```

## 示例

```
【开心】今天天气真好呀，我们一起出去玩吧.wav
【撒娇】人家想要这个嘛，好不好嘛.wav
【难过】为什么要这样对我呢.wav
【感动】谢谢你一直陪在我身边.wav
【调皮】嘿嘿，被我抓到了吧.wav
【普通】今天过得怎么样.wav
```

## 支持的情绪标签

开心、撒娇、调皮、感动、难过、撩拨、普通、惊喜、尴尬、生气、疲惫、积极、疑问、调侃的失望

## 要求

- 格式：WAV
- 采样率：24kHz 或 32kHz（推荐）
- 时长：3~15 秒为宜
- 内容：清晰的语音，无明显背景噪音
